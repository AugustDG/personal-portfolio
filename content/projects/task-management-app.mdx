# Task Management App

A comprehensive task management solution designed for modern teams. This full-stack application combines intuitive task organization with real-time collaboration features, making project management seamless and efficient.

## Project Overview

TaskFlow is a powerful task management platform that bridges the gap between simple to-do lists and complex project management tools. Built with modern web technologies, it provides teams with the tools they need to stay organized and productive.

### Key Features

* **Real-time Collaboration**: Live updates across all connected clients
* **Drag & Drop Interface**: Intuitive task organization with visual feedback
* **Team Workspaces**: Multi-project organization with role-based permissions
* **Advanced Filtering**: Search, sort, and filter tasks by multiple criteria
* **Mobile Responsive**: Full functionality across all device types

## Technical Architecture

### Full-Stack Implementation

The application follows a modern full-stack architecture with clear separation of concerns:

**Frontend (React)**

* Component-based architecture with reusable UI elements

* RESTful API design with Express.js

* Relational database design with Prisma ORM

### State Management

Using Redux Toolkit for predictable state management:

```typescript
// Task slice
const taskSlice = createSlice({
  name: "tasks",
  initialState: {
    items: [],
    loading: false,
    error: null,
    filters: {},
  },
  reducers: {
    setTasks: (state, action) => {
      state.items = action.payload;
    },
    addTask: (state, action) => {
      state.items.push(action.payload);
    },
    updateTask: (state, action) => {
      const index = state.items.findIndex((t) => t.id === action.payload.id);
      if (index !== -1) {
        state.items[index] = { ...state.items[index], ...action.payload };
      }
    },
  },
});
```

  const allowedTypes = ["image/jpeg", "image/png", "application/pdf"];
